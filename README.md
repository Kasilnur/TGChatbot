# 🤖 Иру — Интеллектуальный ИИ-Ассистент

![Python](https://img.shields.io/badge/Python-3.9+-blue?style=for-the-badge&logo=python)
![Telegram](https://img.shields.io/badge/Telegram-Bot%20API-blue?style=for-the-badge&logo=telegram)
![AI](https://img.shields.io/badge/AI-Transformer-orange?style=for-the-badge)

**Иру** — это продвинутый Telegram-бот, построенный на базе современных нейросетевых архитектур (**Transformer**). Он умеет вести осмысленный диалог, понимать голосовые сообщения и анализировать эмоции пользователя.

---



## ✨ Основные возможности
- 🗣 **Голосовое управление**: Отправляй голосовые сообщения — Иру поймет их благодаря интеграции с `SpeechRecognition` и `SoundFile|pydub`.
- 🧠 **Контекстная память**: Бот использует систему "скользящего окна", помня последние 14 сообщений для поддержания связного диалога.
- 🎭 **Эмоциональный интеллект**: Автоматический анализ настроения (Sentiment Analysis) и установка соответствующих реакций (эмодзи).
- ⚡ **Параллельная работа**: Использование многопоточности для обработки запросов без зависаний.

---

## 🛠️ Техническое описание

### 1. Используемый API
Для взаимодействия с мессенджером используется **Telegram Bot API** через библиотеку `pyTelegramBotAPI`. 
- **Описание**: Позволяет отправлять и получать сообщения, обрабатывать голосовые данные, устанавливать реакции на сообщения (Message Reactions) и отображать статус «печатает».
`Hugging Face Inference API`
- **Описание**: Облачный сервис платформы Hugging Face, который позволяет получать доступ к предварительно обученным моделям для различных задач искусственного интеллекта.

### 2. Базовая модель ИИ
В качестве «мозга» бота выступает семейство моделей **Qwen2.5** (версия **72B-Instruct**), доступная через `Hugging Face Inference API`.

### 3. Модель трансформера и решаемые задачи
Проект базируется на архитектуре **Transformer (Self-Attention)**. 
- **Используемая модель**: `Qwen2.5-72B-Instruct`.
- **Решаемые задачи**: 
    - **Генерация ответов**: Построение естественного диалога.
    - **Классификация текста**: Определение эмоционального окраса сообщений.
    - **Обработка контекста**: Анализ истории переписки.

## 🚀 Быстрый запуск

### 1. Требования
Убедитесь, что у вас установлен Python 3.9+ и необходимые библиотеки.

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка окружения
Создайте файл `.env` в корне проекта и добавьте ваши токены:
```env
BOT_TOKEN=ваш_токен_телеграм_бота
DEEPSEEK_TOKEN=ваш_токен_hugging_face
```

### 4. Запуск
```bash
python3 run.py
```
